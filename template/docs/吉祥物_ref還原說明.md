# 吉祥物 `_ref` 姿勢素材 — 換電腦還原說明

製片時 `pipeline\mascot.py` 從 Agent 工作區的 `_ref\` 讀阿遠姿勢圖。
`_ref\` **不在 git repo、不在 D 槽同步範圍**，所以換新電腦後會是空的。
但素材都備份在金主的 Google Drive，照下面複製回來即可。

## 換機還原（最快）

新電腦裝好 Agent 後（部署到 `D:\ayuan-laoshi-agent`），把 Drive 的備份整包複製回 `_ref\`：

```powershell
# Drive 會同步到新機（路徑視帳號而定，通常 G:\我的雲端硬碟）
Copy-Item "G:\我的雲端硬碟\阿遠老師本人\_ref_備份\*.png" "D:\ayuan-laoshi-agent\_ref\" -Force
```

這個 `_ref_備份\` 資料夾裡是**和 mascot.py 對照表一字不差**的 13 個姿勢，複製完製片就能用。

## 想新增/替換姿勢
1. 在 Tensor.art 用 **ayuanff LoRA** + 黃金 prompt 生新姿勢（見 [[ayuan-laoshi-mascot-assets]]）。
2. 白底、單人，存成 `黃色-<姿勢>.png` 放進 `_ref\`，並更新到 `_ref_備份\`。
3. 到 `pipeline\mascot.py` 的 RULES 加對應關鍵字。
4. 缺素材時 `mascot.py` 會自動降級（pick 回 DEFAULT 或空字串），產線不崩。

> 素材原檔與設定表：`阿遠老師本人\`（角色設定表 `阿遠老師角色圖.jpg`＝CHIBA-01、素材庫 `素材庫_正黃色\`）。
